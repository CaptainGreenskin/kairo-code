# Status-line examples

Sample shell scripts for kairo-code's custom-shell status line.

| Script             | Output (with `jq` installed)                        |
|--------------------|-----------------------------------------------------|
| `statusline.sh`    | `[<model> | ctx <pct>%]` color-coded by usage band  |

See [docs/guide/statusline.md](../../docs/guide/statusline.md) for the
full schema + install instructions.
